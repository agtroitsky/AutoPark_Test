Тестовое задание


Требуется автоматизировать и систематизировать отчетность деятельности условного автопарка

 1.  Обязательные объекты и атрибуты:

1.1. Путевые листы:

    № автоматически генерится,
    дата, время выезда,
    время приезда,
    ссылка водитель,
    ссылка автомобиль,
    расход бензина,
    километраж,
    ссылка диспетчер 

1.2. Водители

    ФИО,
    дата рождения 

1.3. Автомобили

    Гос. Номер,
    ссылка модель,
    пробег (должен прибавляться автоматом после каждой поездки),
    год выпуска,
    дата прохождения ТО 

1.4.  Модели авто

    Наименование 

1.5.  Диспетчеры

    ФИО 

 2. БД MSSQL 2000-2016 (Заменена на MySQL 8.0), среда разработки Delphi7-10

 3. Редактирование, добавление, удаление записей с помощью SQL - запросов

 4. Подключения к БД с помощью INI-файла


Дополнительные требования:

    На Главной форме грид с путевками и сверху панель инструментов, должна быть возможность  отбора, сортировки и поиска путевок по любым полям.(компоненты DevExpress приветствуются)
    Редактирование (и добавление) записей должно быть реализовано в модальных формах с автоматическим возвратом по завершении(отмене) на обновленную (новую или исходную при отмене операции) запись.
    При добавлении и редактировании записей реализовать проверку на наличие повторов в БД.
    Записи не удалять, а помечать на удаление, отслеживать ссылочную целостность даже при пометке на удаление.
    Реализовать возможность выгрузки путевок в шаблон word в поля по закладкам и в шаблон excel списка путевых листов.
    Реализовать возможность отображения удаленных путевок с пометкой цветом шрифта в гриде.
    Применение принципов ООП приветствуется.
